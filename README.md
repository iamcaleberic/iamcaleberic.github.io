# iamcaleberic.github.io
##### Portfolio page

 > * Make sure you have `npm` and `node 6.9+` installed before running this
 > * Checkout simple gist to help you Setup [Node Setup](https://gist.github.com/iamcaleberic/12852eb1ec5da2bdd56b9a27036a8ea1)

#### Running
- You can serve on the file API on your browser
- Make sure to clear `Service worker` in `./sw.js` cache for changes to take place.

- Checkout to `app/` and run `npm install` to install dependencies
- Run `npm start`
